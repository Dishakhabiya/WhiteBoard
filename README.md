# eWhiteboard

An interactive digital whiteboard application that allows users to draw, write, and collaborate in real time.

## Features

- ✏️ Freehand drawing
- 🖍️ Multiple colors and brush sizes
- 🧹 Eraser tool
- ↩️ Undo and Redo
- 🗑️ Clear canvas
- 💾 Save drawings
- Export as PNG or PDF
- shapes
- Text tool

## Tech Stack

- React
- JavaScript
- HTML5 Canvas API
- CSS
- WebSockets (Planned)
- Node.js and Express (Planned)

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd ewhiteboard
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```
